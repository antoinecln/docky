Command Line¶

docky¶

Usage:
docky [SWITCHES] [SUBCOMMAND [SWITCHES]] args…
Meta-switches:
-h, --help Prints this help message and quits
--help-all Prints help messages of all sub-commands and quits
-v, --version Prints the program’s version and quits
--verbose Verbose mode
Switches:
-e, --env VALUE
 Environment flag
Sub-commands:  
build Build or rebuild services
down Stop all services
kill Kill all running container of the project
logs View output from containers
migrate TODO Migrate your odoo project
open Open a new session inside your dev container
proxy Start/Restart/Stop your docky proxy
ps List containers
pull Pulls service images
restart Restart service
run Start services and enter in your dev container
up Start all services in detached mode
